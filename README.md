# Pontos_Turisticos_Franca-SP
Portal responsivo sobre os atrativos turísticos de Franca-SP, desenvolvido em HTML5 semântico e CSS3 (Flexbox). O site organiza 9 pontos turísticos em 3 blocos temáticos: Comercial, Esportivo/Lazer e Natureza/Cultura. Possui layout moderno em colunas de cards, design totalmente adaptável para celular.
